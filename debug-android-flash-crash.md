# Debug: android-flash-crash

## 症状
- Android 安装后打开闪退。
- 需要修复后重新生成安装包，并替换图标为 `gemini-svg.svg`。

## 假设
1. `android/main.py` 的 `App.build()` 没有返回根组件，应用启动后立即退出。
2. Android 入口在初始化阶段导入了桌面/平台不兼容代码，导致启动时异常。
3. Buildozer 图标配置未正确指向可用资源，导致打包结果异常或图标未更新。
4. Android 端运行时依赖缺失或过重，导致启动阶段崩溃。

## 计划
- 先修复最可能的启动问题。
- 再替换图标资源并重新打包。
- 最后验证新 APK 的启动行为。
